# fju-sacred-music

## Download Library
1. Download CKFinder from google drive
    - https://drive.google.com/file/d/1oCPDZM4F_nsBIX8CYDxTtUKOcH22DxAT/view?usp=drive_link
2. unzip to path `admin/includes`

## Assign Permission
1. 資料夾需寫入的權限
    - data
    - temp
    - themes
```bash
chmod -R ugo+w data
chmod -R ugo+w temp
chmod -R ugo+w themes
```
